1- to check if an edge(u,v) is used for in any shortest path you can loop the whole nodes and check if dist[u][k] + dist[k][v] <= w so this edge isn't used in any of the shortest pathes here is a code to solve it

for (int k = 1; k <= n; k++) {
    if (k == u || k == v) continue;
    if (f.min_dist[u][k] + f.min_dist[k][v] <= w) {
	   // there is a better path for u and v instead of w so we can remove it
           break;
    }
}
